# iFlyCompass

## 项目简介

**版本：REL1.3.0**

iFlyCompass 是一个多功能的 Web 应用平台，提供了多种实用工具和功能，包括：

- 聊天室功能：支持创建、加入、管理聊天室，实时消息通信
- 小工具集合：提供各种实用的小工具
- 小游戏：内置一些休闲小游戏
- 用户管理：支持用户注册、登录、权限管理
- Passkey 管理：支持生成和管理注册邀请码

## 技术栈

- **前端**：Vue.js、Element UI、原生 JavaScript、CSS3、HTML5
- **后端**：Flask、Python 3.8+
- **数据库**：SQLite
- **实时通信**：Socket.IO
- **认证**：Flask-Login
- **部署**：支持本地运行和打包为 EXE 可执行文件

## 功能特点

### 聊天室功能

- 支持创建带密码和不带密码的聊天室
- 实时消息通信，支持多人聊天
- 显示在线用户列表
- 聊天消息历史记录
- 聊天室管理（编辑、删除）
- 分页显示聊天室列表
- 搜索聊天室功能
- 表情包功能：支持添加、使用和管理表情包
  - 表情商城：浏览和添加公开表情包
  - 表情包管理：管理已添加的表情包
  - 支持单个表情和表情合集
  - 本地缓存表情包，无需网络连接
  - 表情引用功能：可以引用他人发送的表情
  - 表情分类：底部栏分类显示，支持分页切换

### 用户系统

- 用户注册和登录
- 基于 Passkey 的注册邀请机制
- 权限管理（普通用户、管理员、超级管理员）
- 永久会话（除非被其他终端顶号）

### 其他功能

- 必应每日壁纸展示
- 每日诗词推荐
- 响应式设计，支持移动端

## 安装和运行

### 环境要求

- Python 3.8 或更高版本
- pip 包管理工具

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行项目

```bash
python app.py
```

项目将在 `http://127.0.0.1:5002` 上运行。

### 打包为 EXE

```bash
pyinstaller --onefile --name iFlyCompass app.py
```

## 首次使用

1. 启动应用后，访问 `http://127.0.0.1:5002`
2. 点击 "注册" 按钮，创建第一个用户（自动成为超级管理员）
3. 使用创建的账号登录
4. 进入 "Passkey 管理" 页面，生成邀请码
5. 其他用户可以使用邀请码注册
