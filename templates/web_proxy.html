代理控制
代理服务（端口 5003）
基于 mitmproxy，支持 GET/POST 请求，自动重写所有链接和资源
访问
代理地址格式：[[ proxyBaseUrl ]]/<http|https>/<目标网址>
请先开启代理服务，然后输入网址访问
使用说明
示例：访问百度
[[ proxyBaseUrl ]]/https/baidu.com
示例：访问带路径的网址
[[ proxyBaseUrl ]]/https/example.com/path/page
工作原理
内网设备访问代理服务器 → 代理服务器请求外网资源 → 重写所有 URL 和资源引用 → 返回给内网设备
所有 HTML 链接、CSS url()、JS 请求、重定向都会被自动改写为通过代理访问
